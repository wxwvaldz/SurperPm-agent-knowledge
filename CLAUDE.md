# PMPilot 知识库

这是 PMPilot 的共享知识仓库，通过 git 同步实现团队协作。

## 目录说明

- `domain/` — 领域知识（编码标准、格式规范、项目上下文）
- `profiles/` — 团队和个人画像
- `sessions/` — 澄清会话
- `plugins/` — 已安装的插件
- `skills/` — 独立技能
- `mcp/` — MCP 服务器配置
- `learnings/` — 学习记录
- `.logs/` — 业务数据（JSONL，自动管理）

## 注意事项

- 所有文档使用 YAML frontmatter
- `.logs/` 目录由系统自动管理，请勿手动编辑
- 修改后 git push 即可同步到团队
