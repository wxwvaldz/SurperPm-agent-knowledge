# SuperPmAgent Core Skills

| Skill | Path | Use When |
|---|---|---|
| find | `find/SKILL.md` | Discover relevant skills, knowledge, and extensions by convention. |
| find-extensions | `find-extensions/SKILL.md` | Select only the extension prompts relevant to the current action. |
| distill | `distill/SKILL.md` | Turn loop output into reusable skill, knowledge, or extension candidates. |
| normalize-url | `input/normalize-url/SKILL.md` | Convert a URL-like PM input into a normalized session source record before clarification writes IntentSpec files. |
| normalize-feishu-doc | `input/normalize-feishu-doc/SKILL.md` | Convert a Feishu/Lark document link into a specialized session source record before clarification updates IntentSpec files. |
| export-feishu-prd | `output/export-feishu-prd/SKILL.md` | Turn a clarified session into a Feishu PRD artifact record and register the output back into the session. |

These skills define the orchestration layer for SuperPmAgent commands and hooks.
