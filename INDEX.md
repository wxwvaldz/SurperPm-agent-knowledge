# PMPilot 知识库

## 目录结构

| 目录 | 说明 |
|------|------|
| `domain/` | 领域知识（架构、规范、上下文） |
| `profiles/` | 团队和个人画像 |
| `sessions/` | 澄清会话记录 |
| `plugins/` | 已安装的插件 |
| `skills/` | 独立技能定义 |
| `mcp/` | MCP 服务器配置 |
| `learnings/` | 团队学习记录 |
| `extensions/` | 扩展提示注入 |
| `.logs/` | 业务数据（JSONL） |
